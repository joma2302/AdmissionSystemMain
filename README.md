# AdmissionSystem — Приймальна комісія

Веб-додаток для автоматизації роботи приймальної комісії навчального закладу. Система дозволяє реєструвати абітурієнтів, подавати заявки на факультети, ранжувати за балами та проводити зарахування.

## Технології

- **Java 17**, **Spring Boot 3.2.5**
- **Spring MVC** — веб-шар (контролери, REST)
- **Spring Security** — автентифікація, авторизація, ролі (ADMIN / USER)
- **FreeMarker** — шаблонізатор для серверного рендерингу сторінок
- **MySQL** — основна база даних (JDBC, HikariCP)
- **H2 Database** — in-memory база для тестів (режим сумісності з MySQL)
- **Caffeine** — кешування даних
- **Log4j2** — журналювання
- **JUnit 5** — модульне та інтеграційне тестування
- **Maven** — збірка проекту

## Архітектура

Проект побудований за принципами **чистої архітектури** (Clean Architecture):

```
src/main/java/org/example/
├── domain/                  # Доменний шар
│   ├── model/               #   Сутності: Applicant, Faculty, Application, Grade, AdmissionSheet
│   └── repository/          #   Інтерфейси репозиторіїв
├── application/             # Шар застосунку
│   ├── service/             #   Сервіси: ApplicantService, ApplicationService, AdmissionService, AuthService
│   └── dto/                 #   DTO для передачі даних
├── infrastructure/          # Інфраструктурний шар
│   ├── persistence/mysql/   #   MySQL-реалізації репозиторіїв, ініціалізація БД
│   └── web/                 #   Контролери, DTO запитів/відповідей, валідація
├── config/                  # Конфігурація: БД, безпека, кеш
└── Main.java                # Точка входу
```

## Основні можливості

- **Реєстрація абітурієнтів** з оцінками за предметами (шкала 1–12)
- **Управління факультетами** (назва, кількість місць)
- **Подача заявок** на факультет (обмеження: один абітурієнт — один факультет)
- **Ранжування** абітурієнтів за сумою балів
- **Автоматичне зарахування** — визначення зарахованих та відхилених за рейтингом
- **Автентифікація та авторизація** — ролі ADMIN і USER
- **Управління сесіями** — обмеження однієї активної сесії на користувача

## Запуск

### 1. Налаштування бази даних

Створіть базу MySQL та вкажіть параметри підключення у `src/main/resources/application.properties`:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/admission
spring.datasource.username=root
spring.datasource.password=your_password
```

### 2. Збірка та запуск

```bash
mvn clean install
java -jar target/AdmissionSystem-1.0-SNAPSHOT.jar
```

Або через Maven:

```bash
mvn spring-boot:run
```

Додаток буде доступний за адресою: **http://localhost:8090**

## Структура сторінок
