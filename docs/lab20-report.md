2. Провести автоматизоване тестування всіх шарів додатку.

### 3. Лаконічний опис результатів виконання
- **Збірка:** Проект успішно збирається у виконуваний JAR-файл за допомогою команди `mvn clean install`. Усі залежності (Spring Boot, MySQL, FreeMarker, Security) коректно підтягуються.
- **Тестування:** Реалізовано 53 автоматизовані тести, що покривають:
    - Доменні моделі та бізнес-логіку (AdmissionSheet, SubjectRequirement).
    - Шар персистентності (CRUD операції в MySQL).
    - Сервісний шар (AdmissionService, ApplicantService).
    - Веб-контролери адміністративної та клієнтської частини.
- **Профілювання:** Інтегровано Spring Boot Actuator, що дозволяє в режимі реального часу відстежувати стан здоров'я додатку (`/actuator/health`), метрики продуктивності (`/actuator/metrics`) та конфігурацію оточення.

### 4. Текст програми з коментарями

#### Фрагмент конфігурації профілювання (application.properties):
```properties
# Експозиція ендпоінтів Actuator для моніторингу
management.endpoints.web.exposure.include=health,info,metrics,env
# Детальне відображення статусу здоров'я
management.endpoint.health.show-details=always
```
